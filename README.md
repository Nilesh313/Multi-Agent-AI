# 🏥 Multi-Agent Hospital AI System

An intelligent Hospital Agentic AI platform built using FastAPI, SQLite, JWT Authentication, and interactive chatbot workflows.

This project demonstrates:
- Agentic AI concepts
- Chatbot-driven hospital workflows
- Authentication & authorization
- Patient database management
- Interactive frontend UI
- REST APIs using FastAPI

---

# 🚀 Features

## 🤖 AI Chatbot
- Interactive hospital assistant
- Symptom guidance
- Department recommendations
- Appointment booking simulation
- Emergency response handling

## 👨‍⚕️ Patient Management
- Add patients
- View patients
- SQLite local database
- Dummy patient seeding

## 🔐 Authentication
- JWT token-based login
- Protected APIs
- Admin access

## 🌐 Interactive UI
- Chat interface
- Login panel
- Patient dashboard
- Dynamic API interactions

---

# 🛠️ Tech Stack

- HTML/CSS/JavaScript
